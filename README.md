### 🌟 Featured Projects

#### 🩺 VisionDX Mega
> **My Flagship Project** — *🏆 Top 10 finalist, GDG IST Innovators Challenge (Team DEVTECH)*
<br/>
An offline-first AI health diagnostics platform designed specifically for rural and underserved communities in Pakistan. 
It features a massive suite of **12 specialized modules**:
- 👁️ **Diagnostics**: Skin analysis, Eye condition prediction, Hair/scalp analysis.
- 🩹 **Tracking**: Wound tracking using OpenCV, Daily routine tracking.
- 🗣️ **Interactivity**: Cough detection, Lab report analysis, Sleep quality AI, Medicine verification.
- 🤝 **Accessibility**: Pakistan Sign Language (PSL) translation for 71 gestures, VoiceDoc (Voice Interface), Community Health Worker mode, and WhatsApp-based triage.
- **Built with:** React, Vite, Tailwind CSS, Python, Groq LLM, Gemini API, MediaPipe, MySQL, and PHP.

#### 💼 SmartHire
An AI-powered recruitment and applicant tracking tool. Built leveraging the **Gemini API** to streamline candidate screening, automate resume parsing, and drastically improve hiring decisions by reducing bias and saving time.

#### 📡 WiFi Heartbeat Sensing *(Current Research/Build)*
A highly experimental project using WiFi signal variance (RSSI) to detect micro-movements like human heartbeats and breathing patterns through walls or in a localized room. It includes a Python/Flask backend for signal processing (FFT) and a React/Recharts frontend for live continuous monitoring and presence detection.
